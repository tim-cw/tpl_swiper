SwiperJs

L'objectif de cet exercice est de faire un carrousel avec les configurations demandées dans l'exercice

Numéro 1 - Le standard

Aucune configuration spéciale n'est requise. Suivez seulement les étapes suivantes:

  • 1. HTML

    • Ajuster le HTML pour respecter ce que demande la documentation de SwiperJs
    • Ajouter une classe au carrousel ".simple"

  • 2. CSS

    • Charger le SCSS nécessaire pour le fonctionnement de Swiper

  • 3. JavaScript

    • Importer la librairie Swiper dans le fichier Main.js
    • Mettre le code de base pour faire fonctionner Swiper et cibler le carrousel .simple

  • 4. CSS

    • Dans le fichier swiper.scss faite en sorte que .simple prennent tout l'espace
    • Faite en sorte que les images dans .simple prennent tout l'espace

Numéro 2 - Plusieurs slides

Les mêmes configuration que le numéro 1, mais avec plusieurs slides(2) à l'écran en même temps:

  • 1. HTML

    • Ajuster le HTML pour respecter ce que demande la documentation de SwiperJs
    • Ajouter une classe au carrousel ".multiple"

  • 2. JavaScript

    • Mettre le code de base pour faire fonctionner Swiper et cibler le carrousel .multiple
    • Ajouter un paramètre pour avoir 2 slides à l'écran en même temps

  • 4. CSS

    • Dans le fichier swiper.scss faite en sorte que .multiple prennent toute l'espace
    • Faite en sorte que les images dans .multiple prennent toute l'espace

Numéro 3 - Plusieurs cartes

Les mêmes configuration que le numéro 2, mais avec du contenu autre que des images

  • 1. HTML

    • Ajouter la classe .swiper-slide aux cartes(.card)
    • Ajouter une classe au carrousel ".swiper-card"

  • 2. JavaScript

    • Mettre le code de base pour faire fonctionner Swiper et cibler le carrousel .swiper-card
    • Ajouter un paramètre pour avoir 3 slides à l'écran en même temps
    • Ajouter un paramètre pour avoir 30px d'espace entre chaque carte

  • 4. CSS

    • Dans le fichier swiper.scss faite en sorte que .swiper-card prennent tout l'espace
    • Faite en sorte que les images dans .swiper-card prennent tout l'espace

Numéro 4 - Plusieurs cartes

Les mêmes configuration que le numéro 3, mais avec une pagination

  • 1. HTML

    • Ajouter la classe .swiper-slide aux cartes(.card)
    • Ajouter une classe au carrousel ".swiper-pager"
    • Ajouter le html nécessaire pour avoir une pagination

  • 2. CSS

    • Charger le SCSS nécessaire pour la pagination

  • 3. JavaScript

    • Mettre le code de base pour faire fonctionner Swiper et cibler le carrousel .swiper-pager
    • Ajouter un paramètre pour avoir une pagination
    • Ajouter un paramètre pour avoir 3 slides à l'écran en même temps
    • Ajouter un paramètre pour avoir 30px d'espace entre chaque carte

  • 4. CSS

    • Dans le fichier swiper.scss faite en sorte que .swiper-pager prennent tout l'espace
    • Faite en sorte que les images dans .swiper-pager prennent tout l'espace
    • Faite en sorte que la pagination soit sous le carrousel (inspectez l'élément) retirer le positionnement
    • Faite en sorte que les points soit de couleur blanche
