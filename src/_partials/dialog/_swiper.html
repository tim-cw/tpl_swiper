Réusinage

Nous allons maintenant organiser votre JavaScript afin de permettre plusieurs instances de Swiper et préparer nos éventuels prochains numéros.

  • Dans le HTML, ajoutez la classe .js-swiper-base à votre div.swiper.
  • Dans le JS,
    • Créez une méthode initSwiperBase et déplacez votre code d'initialisation de Swiper à l'intérieur de cette méthode. Appelez votre méthode dans votre méthode init() (Regardez comment la méthode init est appellée du constructor).
    • Dans votre méthode initSwiperBase, créez une variable target qui sélectionne dans la page votre élément swiper .js-swiper-base.
    • Dans votre new Swiper(), utilisez votre nouvelle variable target comme cible pour créer le Swiper (premier paramètre).

                initSwiperBase() {
                    const target = document.querySelector('.swiper');
                    const swiper = new Swiper(target, {});
                }

JavaScript

Créez une méthode initSwiperPagination et effectuez les étapes nécessaires pour obtenir une pagination dans le carrousel. Vous pouvez vous inspirer grandement de initSwiperBase.

  • N'oubliez pas de modifier le querySelector de la variable target pour cibler le nouveau carrousel.
  • En vous référant à la documentation de Swiper, ajoutez la configuration nécessaire pour avoir une pagination.
  • Lorsque vous devez transmettre un élément à la pagination, faites un querySelector à partir de la variable target et non du document.

Vous devriez désormais avoir une pagination dans votre carrousel d'images. Il est essentiel de cibler uniquement la pagination de notre carrousel, sinon il cherchera dans toute la page, ce qui pourrait entraîner des problèmes de fonctionnement pour vos carrousels.

Lorsqu'on utilise une variable pour stocker un élément en mémoire et que nous utilisons ensuite cette variable, il est essentiel de s'assurer qu'elle contient bien une valeur existante, sinon cela peut provoquer des erreurs JavaScript. À titre d'exemple, si vous accédez à la deuxième page, vous obtiendrez une erreur car le carrousel n'est pas trouvé. Pour éviter que cela ne provoque des erreurs, nous allons englober notre initialisation de Swiper dans une vérification if(target) {} pour nous assurer que nous avons bien trouvé notre élement HTML swiper.

Cliquez ici pour un autre cadeau. 
                if(target) {
                    const swiper = new Swiper(target, {
                        pagination: {
                          el: target.querySelector('.swiper-pagination'),
                        },
                      });
                }

Style pour la pagination

Vous devez maintenant personnaliser les styles de la pagination. Nous ne souhaitons pas utiliser la couleur bleue par défaut. Utilisez l'inspecteur pour examiner les éléments et les styles créés par SwiperJs, puis écrasez les styles prédéfinis par Swiper pour personnaliser la pagination.

  • Changer la couleur de fond des bullets actifs pour -color-primary (il existe une variable)
  • Changer la couleur de fond des bullets inactifs pour -color-primary (il existe une variable)
  • Modifier la largeur et la hauteur des bullets à 10px (il existe une variable)
  • Modifier le positionnement pour que la pagination soit située en dessous du carrousel.
  • Remplacer les coins arrondis des bullets par des coins carrés.
Cliquez ici pour encore un autre cadeau. 
                .swiper-page {
                    .swiper-slide {
                        background: none;
                        min-height: unset;
                    }
                    .swiper-pagination {
                        //retire le position absolute de d'élément
                        position: static;
                        --swiper-pagination-bullet-width: 10px;
                        --swiper-pagination-bullet-height: 10px;
                        --swiper-pagination-color: var(--color-primary);
                        --swiper-pagination-bullet-inactive-color: var(--color-primary);
                    }
                    .swiper-pagination-bullet {
                        border-radius: 0;
                    }
                }
